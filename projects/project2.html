Bunkify

Bunkify is a web application that allows users to calculate your attendance percentage based on the minimum attendance required, also the number of classes you can bunk and the number of classes you need to attend to reach the minimum attendance percentage. The application is built with html, tailwindcss and Flask. The application is designed in a cartoony theme just to make it look more appealing (have to change the font also to a better one).
