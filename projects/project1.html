Prosthetic Arm

A simple prosethetic arm that can be controlled by giving commands. The arm is controlled by a microcontroller (Ardiuno) and uses servos to move the joints. This project was intially done by a passout senior which we modified so that it can be displayed in a exhibition. The arm is powered by a 9v battery and can run for a long period of time. I really dont have Many pics of the project, just one which i took while the project was being done.

The buidling of Prosthetic arm really was a great experience. It was one of my earliest projects and I learnt a lot from it. The project was done in a team of two consisting me and Nabhan. The project was done in a span of 2 days. I'm planning on creating my own version of the Prosthetic arm in the Future.
